## Changes Made

- **Created** `assets/` directory with three subdirectories:
  - `assets/css/` - Contains all stylesheets
  - `assets/images/` - Contains all images (PNG, JPG)
  - `assets/js/` - Contains all JavaScript files

- **Moved** all asset files from root to organized folders
- **Updated** all file references in `index.html` to point to new paths
